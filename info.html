Question-2: What is the difference between innerHTML, innerText, and textContent ?

  • innerHTML retrieves the complete HTML content within an element, including tags.
  • innerText only returns the visible text content of an element, ignoring HTML tags and hidden elements
  • textContent retrieves all text content within an element, including hidden text, but without considering HTML tags or styling

Question-3: What is event delegation in the DOM?

Event delegation is a technique where i attach a single event listener to a parent element instead of adding listeners to multiple child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a type of DOM event propagation where the event first triggers on the innermost target element
